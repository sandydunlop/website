    assembleTextAndLinks

    public static void assembleTextAndLinks(Api a, Context c)

    Generates Text objects for links to types and links in Javadoc text.This is where we decide if the label for those links shows qualified names or simplified names.

    Parameters:

    a - The API model

    c - The doclet context to keep track of why package and type are being processed
